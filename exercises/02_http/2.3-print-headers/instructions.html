Print request headers

Exercise overview

In this exercise you will create a Node server, which will respond to any incoming requests by sending the headers object from that request as plain text.

Implementing the exercise

Start by making sure you are in your local clone of the Git repository, and in its exercises/02_http/2.3-print-headers directory where this instructions.html file is placed. Then:

  • In this directory run the command npm init to initialize your Node project. Use the default values offered by npm by pressing Enter.
  • Write your server code in index.js file you create inside this directory
  • In your server code use createServer-method to create the server
  • Use response's writeHead-method to set the correct HTTP response status code (200) and content type (text/plain) for the response.
  • Inside createServer method's callback, use the Node.js's built-in http module's message.headers method to access the HTTP headers from the incoming message.
  • As the headers from message.headers are an object, they cannot be sent as-is, but you need to use JSON.strigify JavaScript method to convert the object to a string
  • Use response's write method to add the header string to the response
  • Use response's end method to end adding to the response, and send it to the client
  • You can run your server by running the command node index.js in this directory
  • Correct all possible mistakes
  • Commit your index.js and package.json files, and push them to your Git repository.
