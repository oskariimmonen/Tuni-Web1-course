3.4 DOM Tree

Exercise Overview

In this exercise you are going to use JavaScript to work with the DOM tree of an HTML document. This includes creating and deleting elements and manipulating element attributes. The index.html file includes the starting HTML of this exercise and your job is to write the accompanying JavaScript and save it to dom-tree.js file inside the same directory.

Implementing the Exercise

Start by making sure you are in your local clone of the Git repository, and in its exercises/03_html/3.4-dom-tree directory where this instructions.html file is located. Then:

  • Write your code into dom-tree.js file in this directory.
  • You can find the HTML in the index.html file inside this directory. Make note of the IDs and classes of the elements in the HTML. You should use them in your code to get hold of the elements. DO NOT EDIT THIS FILE!
  • Your code should perform the following DOM manipulations:
    1. Add a class list to the navigation link list (unordered list, ul) inside nav element.
    2. Create a new link (anchor tag, a) as the last item inside the navigation link list. The link must have http://localhost:3000/ as its target address (href attribute) and the text Localhost inside. Remember to put the anchor tag inside a list item (li) tag.
    3. Insert a new list item (li tag) with the text Item 0 as the first element of the ordered list (ol) with the id ordered. Note that the text has the number zero as its last character.
    4. Remove the class navi from the todo list (unordered list, ul, with the id em{todo}). Do not remove any other classes from the list!
    5. Delete the second list item from the todo list. The the text inside the item is Item 2.
    The order of the DOM manipulations inside your code does not matter. Only the final resulting DOM tree is important.

Remember to commit dom-tree.js and push the changes to your Git repository before you submit your work in Plussa.
